showdown

A PIP8 compliant tool ready to automate the network verification checks of local Cisco iOS devices. Designed to increase productivity within a live Amazon site during the project review of a site's network infrastructure.

  1. Follow the README to install on Windows using PIP package manager.
  2. Enter your device credentials and destination host name into the prompt.
  3. Showdown will create a new folder on your desktop named, "Network Checks".
  4. Showdown SSH's into the destination device, runs a set of commands, and saves the command output to a 'file.txt' named after the device.
  5. The file is stored within your new desktop folder.
  6. You will be asked to enter "y" to continue the SSH process for any additional destination hosts during the session. Otherwise, the program will terminate.
