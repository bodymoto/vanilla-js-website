Projects

Showdown

A tool to automate local network health checks on Cisco iOS devices. Enter your credentials and device host name into Showdown's prompt. A new folder will be created on your Windows desktop called, "Network Checks". Showdown then SSH's into the device, runs a set of commands, and saves the command output within a text file named after the device. This file is saved inside the desktop folder. Showdown will then prompt for another host name and repeat the SSH process, or stop running when "q" is entered.

➪Written in Python. Use the PIP package manager to install. Windows 10 tested
